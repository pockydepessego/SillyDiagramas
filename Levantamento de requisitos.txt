Levantamento de Requisitos

RF1. O sistema deverá permitir efetuar registo ao utilizador.
RF2. O sistema deverá permitir o log-in e o log-out dos utilizadores.
RF3. O sistema deverá permitir a gestão CRUD do utilizador. (CRUD- Create, Read, Update, Delete.)
RF4. O sistema deverá permitir compra de tokens.
RF5. O sistema deverá permitir consulta dos tokens.
RF6. O sistema deverá permitir o cálculo do consumo de tokens.
RF7. O sistema deverá permitir a gestão CRUD dos projetos.
RF8. O sistema deverá permitir fazer o upload de diagramas nos formatos especificados.
RF9. O sistema deverá permitir ao utilizador a gestão dos seus diagramas.
RF10. O sistema deverá permitir a criação e setup de um projeto.
RF11. O sistema deverá permitir uma integração com vários repositórios de código diferentes.
RF12. O sistema deverá permitir realizar uma análise e estimativa de custos do projeto.
RF13. O sistema deverá permitir gerar o código para o utilizador através do AI.
RF14. O sistema deverá permitir a revisão e validação do código pelo utilizador.
RF15. O sistema deverá permitir consultar estatísticas de utilizador.
RF15. O sistema deverá permitir enviar notificações ao utilizador em cada etapa.
RF16. O sistema deverá permitir desativar/ativar notificações.
